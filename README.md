# Pogudina Studio

Portfolio site for Valentina Pogudina, focused on frontend engineering, UI architecture, accessibility, and interactive product work.

Live site: `https://baabochka.github.io/pogudina-studio/`

## Overview

This project is a custom portfolio built to present case studies, selected product work, and interactive browser-based games in a format that feels polished, lightweight, and intentional.

The site combines editorial-style project storytelling with hands-on frontend experiments, including SVG-driven game interfaces and responsive UI systems.

## Built With

- React
- TypeScript
- Vite
- Tailwind CSS v4
- React Router

## Highlights

- Case-study driven project pages with structured product, technical, and design context
- Interactive games section for puzzle and UI experiments
- Reusable UI primitives and shared design tokens
- GitHub Pages deployment
